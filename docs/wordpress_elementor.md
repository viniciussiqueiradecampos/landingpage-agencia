# Documentação WordPress e Elementor

Este documento contém os recursos essenciais para o desenvolvimento da Landing Page utilizando WordPress e Elementor.

## 1. WordPress (O CMS)
O WordPress será utilizado como a base para a gestão de conteúdo.
- **Site Oficial:** [https://wordpress.org/](https://wordpress.org/)
- **Documentação para Desenvolvedores:** [https://developer.wordpress.org/](https://developer.wordpress.org/)
- **Referência de Código:** [https://developer.wordpress.org/reference/](https://developer.wordpress.org/reference/)

### Instalação Recomendada
Para Windows, recomendamos o uso do **[LocalWP](https://localwp.com/)**. É a ferramenta mais simples e performática para rodar WordPress localmente sem precisar configurar Apache/MySQL manualmente.

## 2. Elementor (Page Builder)
O Elementor é o construtor visual preferencial para este projeto.
- **Site Oficial:** [https://elementor.com/](https://elementor.com/)
- **Documentação para Desenvolvedores:** [https://developers.elementor.com/](https://developers.elementor.com/)
- **Guia de Widgets Personalizados:** [https://developers.elementor.com/docs/widgets/](https://developers.elementor.com/docs/widgets/)

### Plugins Essenciais
Para uma Landing Page de Agência, recomendamos os seguintes plugins:
1. **Elementor:** Construtor principal.
2. **Elementor Header & Footer Builder:** Para personalizar cabeçalhos e rodapés.
3. **WPForms** ou **Contact Form 7:** Para captura de leads.
4. **All-in-One WP Migration:** Para facilitar o deploy futuro.
5. **Yoast SEO** ou **Rank Math:** Para otimização de busca.
